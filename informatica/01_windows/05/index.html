Cronograma: Proteção e Manutenção do Sistema

Aprenda a proteger, otimizar e gerenciar o funcionamento do computador.

0:00 - 0:20 | Antivírus

  • Vídeo: A Defesa
    • O vídeo será passado em partes, explicando a importância de manter um antivírus atualizado, como ele detecta ameaças e protege o sistema contra vírus, malwares e outras ameaças.

0:20 - 0:40 | Proteção dos Computadores

  • Windows Defender:
    • O Windows Defender é uma solução integrada no Windows para proteção contra vírus e ameaças. Vamos explorar como ativá-lo, verificar atualizações e realizar verificações periódicas.
  • Firewall:
    • O Firewall protege contra acessos não autorizados à rede. Serão demonstrados passos para configurá-lo no painel de controle e criar regras de bloqueio e liberação de programas.

0:40 - 1:00 | Impressoras

  • Instalar Drivers:
    • Aprenda a localizar e instalar os drivers adequados para o modelo da impressora, seja manualmente ou utilizando ferramentas automáticas.
  • Compartilhar na Rede:
    • Configuração de uma impressora para ser acessível na rede local. Inclui a ativação do compartilhamento e ajustes no Windows para permitir o uso por outros dispositivos.

1:00 - 1:20 | Programas

  • Instalações de Programas:
    • Guia para instalação segura de softwares, verificando a procedência e evitando a instalação de adwares ou programas indesejados.
  • Desinstalações de Programas:
    • Passo a passo para desinstalar programas corretamente, limpando arquivos residuais e otimizando o desempenho do sistema.

1:20 - 1:40 | Limpeza de Disco

  • Limpeza de Disco:
    • Uso da ferramenta "Limpeza de Disco" para remover arquivos temporários, cache e outros dados desnecessários, liberando espaço no sistema.

1:40 - 2:00 | Restauração do Sistema

  • Criar Ponto de Restauração:
    • Demonstrar como criar um ponto de restauração no Windows para salvar o estado atual do sistema, permitindo reverter alterações futuras.
  • Usar a Restauração do Sistema:
    • Explicação prática de como restaurar o sistema para um ponto anterior em caso de problemas ou mau funcionamento.
