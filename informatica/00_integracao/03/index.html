Cronograma: Conhecendo os Principais Componentes do Computador

Explore os componentes essenciais de um computador e entenda como cada parte contribui para o seu funcionamento.

0:00 - 0:15 | Monitor

  • O que é o Monitor?
    • O monitor é o componente de saída do computador que exibe as informações visuais processadas pela CPU. Ele pode ter diferentes tamanhos, resoluções e tecnologias (LED, LCD, OLED).
    • É através do monitor que os usuários interagem visualmente com o sistema operacional e os programas.
    • Exemplo de Tipos: Monitores de tela plana, curvas, ultra-wide e de alta resolução (4K, Full HD, etc.).
  • Discussão: Tipos de monitores e suas aplicações no dia a dia.

0:15 - 0:30 | Mouse

  • O que é o Mouse?
    • O mouse é um dispositivo de entrada utilizado para interagir com o computador. Ele permite apontar, clicar e arrastar objetos na tela.
    • Existem diferentes tipos de mouse: mecânico (com bolinha), óptico, laser e sem fio (wireless).
    • Funções Principais: Selecionar, abrir arquivos, arrastar objetos, navegar por páginas e muito mais.
  • Atividade Prática: Utilização do mouse para interagir com aplicativos simples e jogos educativos.

0:30 - 0:50 | Teclado

  • O que é o Teclado?
    • O teclado é o principal dispositivo de entrada para digitação de textos, números e comandos no computador.
    • Ele é composto por teclas alfanuméricas, teclas de controle (como enter, backspace), teclas de função e teclas especiais.
    • Tipos de Teclados: Teclados mecânicos, de membrana, ergonômicos, compactos, entre outros.
  • Atividade Prática: Exercícios de digitação para familiarização com o layout do teclado.

0:50 - 1:05 | Gabinete

  • O que é o Gabinete?
    • O gabinete é a estrutura que abriga todos os componentes internos do computador, como a placa-mãe, a CPU, o HD, entre outros.
    • Ele serve para proteger os componentes e permitir uma organização interna que facilita a circulação de ar e a manutenção do sistema.
    • Tipos de Gabinetes: ATX, micro-ATX, mini-ITX, entre outros, variando em tamanho e capacidade de armazenamento.
  • Discussão: Como escolher um bom gabinete de acordo com as necessidades do sistema.

1:05 - 1:25 | CPU (Unidade Central de Processamento)

  • O que é a CPU?
    • A CPU é o "cérebro" do computador. Ela processa os dados e executa as instruções de programas e aplicativos.
    • A CPU é composta por núcleos, que são unidades responsáveis por realizar as tarefas de processamento de dados.
    • Exemplo de Processadores: Intel, AMD, com modelos variando entre quad-core, hexa-core, octa-core, etc.
  • Atividade Prática: Identificar a CPU em computadores reais e verificar suas especificações.

1:25 - 1:40 | HD (Disco Rígido)

  • O que é o HD?
    • O HD é um dispositivo de armazenamento de dados permanente do computador. Ele armazena o sistema operacional, programas, arquivos de dados, entre outros.
    • Existem dois tipos principais de HD: HDD (disco rígido tradicional) e SSD (unidade de estado sólido), sendo o SSD mais rápido e eficiente.
    • Capacidade de Armazenamento: Pode variar de 120GB a vários terabytes, dependendo do tipo e uso do computador.
  • Discussão: Vantagens do SSD sobre o HD tradicional em termos de desempenho.

1:40 - 1:55 | Memórias

  • O que são Memórias?
    • As memórias são responsáveis por armazenar temporariamente dados e instruções que a CPU precisa acessar rapidamente. Elas influenciam diretamente o desempenho do computador.
    • Tipos de Memórias: RAM (memória de acesso aleatório) e ROM (memória somente leitura).
    • A memória RAM é volátil, ou seja, perde os dados quando o computador é desligado. A memória ROM contém informações essenciais para o funcionamento básico do computador.
  • Discussão: Como a quantidade de memória RAM afeta o desempenho do computador.
