Cronograma: Texto e PowerClip no CorelDRAW

Aprenda a trabalhar com texto e a usar o recurso PowerClip para criar designs mais dinâmicos.

0:00 - 0:40 | Ferramenta Texto

  • Texto Artístico:
    • Explanação sobre o uso do texto artístico, ideal para títulos e palavras com estilo criativo.
    • Exemplos práticos:
      • Adicionar texto diretamente na tela com a ferramenta Texto.
      • Aplicar efeitos como distorção, sombreamento e preenchimento ao texto.
  • Texto de Parágrafo:
    • Introdução ao texto de parágrafo, usado para blocos de texto extensos, como descrições e textos explicativos.
    • Prática:
      • Criar uma caixa de texto ajustável e editar alinhamentos, espaçamento entre linhas e margens.
      • Aplicar preenchimentos e contornos ao texto dentro da caixa.

0:40 - 1:20 | Exercício Prático: Texto

  • Objetivo:
    • Os alunos criarão um pôster com título estilizado usando texto artístico e informações adicionais em texto de parágrafo.
    • Serão incentivados a explorar efeitos de contorno, sombra e diferentes fontes.

1:20 - 2:00 | PowerClip

  • Introdução ao PowerClip:
    • O recurso PowerClip permite colocar objetos dentro de outros para criar composições criativas.
    • Exemplos de uso comum, como inserir imagens ou padrões dentro de formas vetoriais.
  • Prática com PowerClip:
    • Demonstrar como inserir objetos dentro de formas com o comando *PowerClip Dentro*.
    • Editar o conteúdo do PowerClip e ajustar posicionamentos.
  • Exercício Prático:
    • Criar um design personalizado usando PowerClip para inserir uma imagem dentro de uma forma complexa.
    • Adicionar texto artístico ao design para compor um cartaz ou peça publicitária.
