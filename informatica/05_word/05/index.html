Cronograma de Edição no Microsoft Word

Aprenda a usar o Guia Correspondências para criar mala direta, etiquetas e envelopes personalizados de forma eficiente.

0:00 - 0:20 | Mala Direta: Assistente de Mala Direta Passo a Passo

  • Assistente de Mala Direta: O Microsoft Word possui um assistente que facilita a criação de mala direta. A primeira etapa é iniciar um novo documento e ir até a guia "Correspondências" para acessar o "Assistente de Mala Direta". Este assistente guia o usuário por cada passo necessário, desde a criação da carta até a finalização do envio para os destinatários.
  • Passos principais do Assistente:
    • Escolher o Tipo de Documento: O usuário pode escolher entre cartas, envelopes, etiquetas ou mensagens de e-mail.
    • Selecionar o Banco de Dados: Escolha uma fonte de dados, como uma planilha do Excel, contendo as informações dos destinatários.
    • Inserir Campos de Mesclagem: Durante o processo, é possível inserir campos personalizados (como nome, endereço, etc.) no documento. O Word substituirá automaticamente essas variáveis pelos dados reais de cada destinatário.
    • Pré-visualizar e Concluir: O assistente permite a pré-visualização do documento para garantir que todos os dados sejam inseridos corretamente antes de realizar a mesclagem.

0:20 - 0:40 | Mala Direta: Usar um Banco de Dados Externo (usando Excel)

  • Uso de Banco de Dados Externo: O Word permite utilizar bancos de dados externos para a mala direta, como planilhas do Excel. Essas planilhas devem estar organizadas com colunas para os diferentes campos de dados, como "Nome", "Endereço", "Cidade", etc. O Word se conecta a essas planilhas para inserir os dados no documento de mala direta.
  • Como vincular o Excel ao Word: O usuário deve selecionar "Selecionar Destinatários" e, em seguida, "Usar Lista Existente" para localizar a planilha do Excel. Após isso, será possível escolher qual coluna do Excel corresponde a qual campo de dados na mala direta.
  • Exemplo Prático: Se você tem uma lista de clientes em Excel, ao vincular essa planilha ao documento de mala direta no Word, você pode criar uma carta personalizada para cada cliente, preenchendo automaticamente os dados como nome e endereço.

0:40 - 1:00 | Etiquetas no Word

  • Criação de Etiquetas: O Word permite criar etiquetas personalizadas utilizando a guia "Correspondências" e a opção "Etiquetas". Primeiro, é preciso escolher o tipo de etiqueta (baseado no modelo de etiqueta que você comprou) e configurar o tamanho. O Word se ajusta automaticamente ao modelo de etiqueta e permite a personalização do texto.
  • Personalizando Etiquetas: Dentro de cada etiqueta, você pode adicionar texto, imagens e até campos de mesclagem para criar etiquetas personalizadas com nomes e endereços. Isso é ideal para situações como correspondências em massa ou envio de pacotes.
  • Mesclagem de Dados para Etiquetas: Assim como na mala direta, você pode usar um banco de dados externo (como uma planilha do Excel) para preencher os campos de mesclagem nas etiquetas. O Word fará a substituição automática dos dados para cada etiqueta, economizando tempo e evitando erros de digitação.

1:00 - 1:20 | Envelopes no Word

  • Criação de Envelopes: O Word também permite criar envelopes personalizados para correspondência. A funcionalidade de criação de envelopes está na guia "Correspondências", onde você pode selecionar "Envelopes" e preencher as informações do destinatário e do remetente.
  • Configuração de Envelopes: Para configurar o envelope, você pode escolher o tamanho (por exemplo, DL ou C6), a posição do endereço de envio e o layout do remetente. É possível ajustar o envelope para corresponder à sua impressão ou àquilo que a empresa ou organização exige.
  • Impressão de Envelopes: Após configurar o envelope, você pode imprimir os envelopes individualmente ou em uma série de documentos, o que é útil para envios em massa. O Word permite configurar a impressão para que os endereços sejam automaticamente colocados no lugar correto de cada envelope.

1:20 - 1:40 | Prática: Criação de Mala Direta, Etiquetas e Envelopes

  • Prática de Mala Direta: Os alunos serão orientados a usar a ferramenta de mala direta para criar um documento personalizado (como uma carta) para uma lista de contatos armazenada em uma planilha Excel. A prática envolve a escolha do banco de dados e a inserção de campos de mesclagem para gerar um documento único para cada destinatário.
  • Prática de Etiquetas: Os alunos criarão etiquetas para um conjunto de endereços. Eles aprenderão a vincular uma planilha de Excel e personalizar as etiquetas de forma eficiente.
  • Prática de Envelopes: Por fim, os alunos criarão envelopes personalizados e configurados para correspondência, aprendendo a configurar o layout e imprimir os envelopes de forma prática e rápida.
