Cronograma: Microsoft PowerPoint

Aprenda a usar o Microsoft PowerPoint para criar apresentações, trabalhar com slides e explorar suas funcionalidades.

0:00 - 0:20 | Exemplos de Apresentações Prontas

  • Exemplos de Apresentações Prontas:
    • Exploração de exemplos prontos de apresentações que podem ser utilizados como base para suas próprias criações.
    • Exibição de diferentes tipos de apresentações: educacional, corporativa, pessoal, etc.
    • Discussão sobre como adaptar esses modelos de acordo com as necessidades do usuário.

0:20 - 0:40 | Microsoft PowerPoint: Apresentação do Ambiente

  • Relembrar Guias e Grupos:
    • Exploração das principais guias do PowerPoint, como "Arquivo", "Página Inicial", "Inserir", "Design", "Transições", "Animações", etc.
    • Como cada guia é organizada em grupos de comandos, facilitando o acesso às ferramentas necessárias para cada etapa da criação de uma apresentação.
  • Zoom:
    • Como utilizar a ferramenta de zoom para ampliar ou reduzir a visualização dos slides, facilitando a edição de detalhes.
  • Barra de Status:
    • Exploração da barra de status na parte inferior da tela, que oferece informações sobre o slide atual, o modo de exibição, entre outros detalhes importantes.
  • Modos de Exibição de Apresentação:
    • Explicação dos diferentes modos de exibição no PowerPoint, como a "Vista Normal", "Vista de Slides", "Vista de Classificação de Slides" e "Vista de Apresentação de Slides".
  • Adicionar Anotações:
    • Como adicionar anotações ao slide para auxiliar durante a apresentação, sem que as notas apareçam na projeção do slide.

0:40 - 1:00 | Modelos Prontos do Office

  • Modelos Prontos do Office:
    • Como utilizar os modelos prontos oferecidos pelo Microsoft PowerPoint para agilizar a criação de apresentações.
    • Exploração de modelos de slides corporativos, educacionais, informativos, entre outros.
    • Como personalizar esses modelos para atender às necessidades do usuário, alterando textos, imagens e estilos.

1:00 - 1:20 | Slide: Manipulação Básica

  • Criar Novo Slide:
    • Como criar um novo slide utilizando a opção de "Novo Slide" na guia "Página Inicial" ou o atalho "Ctrl+M".
  • Escolher Layout de Slide:
    • Exploração dos diferentes tipos de layout de slides (Título, Título e Conteúdo, Comparação, etc.) e como escolher o layout apropriado para cada slide.
  • Redefinir Layout de Slide:
    • Como redefinir o layout de um slide para o layout original, caso você queira mudar a estrutura do slide depois de já ter criado.
  • Excluir e Duplicar Slides:
    • Como excluir slides desnecessários e como duplicar slides existentes, usando o menu de contexto ou atalhos de teclado (Ctrl+D).

1:20 - 1:40 | Estrutura de Tópicos e Salvamento

  • Estrutura de Tópicos (TAB / SHIFT+TAB):
    • Como usar as teclas "TAB" e "SHIFT+TAB" para mover para os níveis superiores ou inferiores dentro de uma lista de tópicos, facilitando a organização do conteúdo.
  • Salvar e Salvar Como:
    • Como usar a opção "Salvar" para salvar seu progresso regularmente, e "Salvar Como" para criar versões diferentes do seu arquivo em locais ou formatos distintos (PPT, PDF, etc.).
