Cronograma: Guia Inserir e Animações no PowerPoint

Aprenda a inserir elementos gráficos e aplicar animações no Microsoft PowerPoint para criar apresentações dinâmicas e visuais.

0:00 - 0:20 | Guia Inserir: WordArt e Formas

  • WordArt:
    • Cor: Como alterar a cor do WordArt para se adequar ao estilo da apresentação, utilizando a paleta de cores ou definindo cores personalizadas.
    • Fonte: Como mudar o tipo, tamanho e estilo da fonte do WordArt para tornar o texto mais atraente e legível.
    • Formas: Como aplicar formas no WordArt para destacar palavras e frases, como sombreamento ou contorno, para efeitos visuais.
    • Efeitos: Como adicionar efeitos ao WordArt, como sombras, reflexos e 3D, para criar um impacto visual maior na apresentação.
  • Formas:
    • Preenchimento: Como preencher formas com cores sólidas, gradientes, padrões ou texturas para tornar as formas mais visíveis.
    • Contorno: Como alterar o contorno das formas, ajustando a cor, espessura e estilo da linha para melhorar a apresentação.
    • Editar Pontos: Como editar as formas manipulando seus pontos, para criar formas personalizadas e ajustadas às necessidades da apresentação.
    • Grupo Organizar:
      • Avançar e Recuar: Como organizar a ordem das formas, enviando para frente ou para trás em relação a outros objetos.
      • Girar: Como girar as formas para posicioná-las da forma desejada.
    • Escrever Dentro da Forma: Como inserir texto dentro de formas para criar botões ou caixas de texto que se ajustem ao formato da forma.

0:20 - 0:40 | Inserindo Imagens

  • Relembrar Conceitos de Correções:
    • Como realizar ajustes em imagens inseridas no PowerPoint, utilizando as ferramentas de correção, brilho, contraste e nitidez.
    • Como aplicar efeitos de cor, como saturação, matiz e tonalidade, para melhorar a aparência das imagens.
    • Como cortar imagens para destacar a parte mais importante da imagem ou ajustar o tamanho conforme necessário.
  • Remover Plano de Fundo:
    • Como utilizar a ferramenta "Remover plano de fundo" para eliminar o fundo de uma imagem, deixando apenas o objeto principal visível, o que é útil para criar efeitos visuais interessantes.

0:40 - 1:00 | Guia Animações: Tipos e Aplicação

  • Entrada:
    • Como aplicar animações de entrada a objetos e textos, como Fade, Aparecer, Voar, etc., para que eles apareçam de forma dinâmica na apresentação.
  • Ênfase:
    • Como adicionar animações de ênfase, como Pulsar, Crescer/Diminuir, para destacar objetos ou textos durante a apresentação.
  • Saída:
    • Como aplicar animações de saída para fazer objetos desaparecerem de forma impactante, como desaparecer gradualmente ou voar para fora da tela.
  • Caminho de Animação:
    • Como aplicar um caminho de animação personalizado a um objeto, permitindo que ele se mova ao longo de um trajeto definido na tela.
    • Exploração das opções de caminhos de animação predefinidos, como linha reta, arco, e curvas.

1:00 - 1:40 | Prática e Aplicação dos Conceitos

  • Atividade Prática:
    • Os alunos irão criar um slide com diferentes formas, WordArt, e animações de entrada e saída. Também deverão inserir imagens, aplicar correções e remover planos de fundo das imagens inseridas.
    • Durante a atividade, o professor irá supervisionar a aplicação dos conceitos, oferecendo feedback e orientações conforme necessário.

1:40 - 2:00 | Dúvidas e Conclusão

  • Dúvidas:
    • Os alunos poderão esclarecer quaisquer dúvidas sobre o uso das ferramentas de inserção, animação e edição de imagens no PowerPoint.
  • Revisão:
    • Revisão geral do conteúdo abordado na aula, destacando as principais funcionalidades do PowerPoint para criar apresentações dinâmicas e visualmente atraentes.
